    Etiquetas Multimedia

    Estas etiquetas de multimedia permiten incrustar y reproducir fácilmente contenido multimedia, como imágenes, audio y video, en páginas web de manera estándar y compatible con diferentes navegadores y dispositivos.

    img Esta etiqueta se utiliza para insertar imágenes en una página web. La etiqueta img no tiene una etiqueta de cierre y utiliza el atributo src para especificar la ruta de la imagen y el atributo alt para proporcionar un texto alternativo que describe la imagen.
    audio Se utiliza para insertar contenido de audio en una página web. Puedes especificar la fuente del archivo de audio utilizando el atributo src y proporcionar controles de reproducción utilizando la etiqueta audio o JavaScript.
    video Se utiliza para insertar contenido de video en una página web. Al igual que audio, puedes especificar la fuente del archivo de video utilizando el atributo src y proporcionar controles de reproducción utilizando la etiqueta video o JavaScript.
