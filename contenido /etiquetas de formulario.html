    Etiquetas de Formulario

    Estas etiquetas se utilizan para crear formularios interactivos en HTML, que permiten a los usuarios enviar datos al servidor web para procesamiento adicional.

    Form Define un formulario en HTML. Esta etiqueta encierra todos los elementos de formulario, como campos de entrada, botones y etiquetas.
    Input Define un campo de entrada en un formulario. El tipo de campo (por ejemplo, texto, contraseña, radio, checkbox) se especifica mediante el atributo type.
    Select Define un menú desplegable de opciones en un formulario. Puede contener uno o más elementos option que representan las opciones seleccionables.
    option Define una opción seleccionable dentro de un menú desplegable
    Label Define una etiqueta descriptiva para un elemento de formulario. Al hacer clic en la etiqueta, se enfoca el campo de entrada asociado.
