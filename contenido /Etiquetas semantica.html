Conocimiento De HTML5

    Etiquetas Semantica
    Header Define la cabecera de un documento o una sección
    Nav Define una sección de navegación
    Main Define el contenido principal de un documento
    Article Define un contenido independiente y autónomo, como un artículo, una entrada de blog o un comentario
    Figure Define contenido multimedia, como imágenes, gráficos, videos, etc., con una leyenda opciona
    Footer Define el pie de página de un documento o una sección
    Section Define una sección genérica de un documento
    Aside Define contenido secundario, como una barra lateral o un contenido relacionado.
