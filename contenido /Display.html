Display

    Display s
    block Un elemento con display: block ocupa todo el ancho disponible y comienza en una nueva línea, lo que significa que ocupa todo el ancho del contenedor y se muestra en una línea separada de otros elementos.
    inline Un elemento con display: inline ocupa solo el ancho necesario y no comienza en una nueva línea, lo que significa que se ajusta al contenido y permite que otros elementos aparezcan en la misma línea que él.
    inline-block Combina las características de los elementos block e inline. Un elemento con display: inline-block ocupa solo el ancho necesario como un elemento inline, pero permite establecer alturas y anchuras como un elemento block
    none Oculta el elemento y lo quita del flujo del documento. Los elementos con display: none no ocupan espacio en la página y no son renderizados por el navegador.
    table Transforma un elemento en un elemento de tabla. Este valor permite que el elemento se comporte como una tabla HTML, con las características de diseño asociadas a las tablas
