    Etiquetas de Texto
    h1, h2, h3, h4, h5, h6 Estas etiquetas se utilizan para definir encabezados de diferentes niveles, donde h1 es el más importante y h6 el menos importante. Se utilizan para organizar y estructurar el contenido de la página.
    p p Esta etiqueta se utiliza para definir un párrafo de texto. Se utiliza para dividir el contenido en bloques de texto coherentes y legibles.
    span Esta etiqueta se utiliza para aplicar estilos u otras características a partes específicas de texto dentro de un elemento más grande, sin afectar la estructura del documento.
    strong Esta etiqueta se utiliza para resaltar o enfatizar el texto, generalmente mostrándolo en negrita. Indica que el texto es especialmente relevante o importante.
    em Esta etiqueta se utiliza para enfatizar el texto, generalmente mostrándolo en cursiva. Se utiliza para resaltar partes del texto que son especialmente significativas o importantes.
    a Esta etiqueta se utiliza para crear un enlace hipertexto a otra página web, archivo o ubicación dentro del mismo documento. Se utiliza para navegar entre diferentes recursos en la web.
    br Esta etiqueta se utiliza para insertar un salto de línea en el texto, lo que provoca que el texto siguiente aparezca en una nueva línea.
