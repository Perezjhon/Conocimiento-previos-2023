Conocimiento De GITHUB

    GITHUB
    git init crea un nuevo repositorio de Git.
    git add. Mueve los cambios del directorio de trabajo al área del entorno de ensayo.
    git config --global user.name es una función útil que sirve para definir valores de configuración del nombre
    git config --global user.email es una función útil que sirve para definir el correo electronico
    git status muestra el estado del directorio de trabajo
    git commit -m "comentario" permite hacer instantáneas del estado actual del trabajo y guardar los cambios.
    git brach permite ver la rama
    git config user.email permite ver el correo electronico
    git config user.name permite ver la configuración que se hizo
    git log Te permite ver el historial del proyecto, filtrarlo y buscar cambios concretos.
    git checkeout -b permite cambiar las rama
    git remote add origin permite crear, ver y eliminar conexiones con otros repositorios.
