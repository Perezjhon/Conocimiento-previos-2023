    Etiquetas de Agrupacion

    Estas etiquetas de agrupación son fundamentales para organizar y estructurar el contenido de una página web de manera significativa, lo que facilita su comprensión, mantenimiento y manipulación mediante CSS y JavaScript.

    div Esta etiqueta se utiliza como un contenedor genérico para agrupar elementos y aplicar estilos o scripts a ellos. Esencialmente, divide el contenido en bloques independientes que pueden ser manipulados de forma conjunta.
    span Aunque técnicamente es una etiqueta de texto, también se puede considerar una etiqueta de agrupación cuando se utiliza para envolver partes específicas de texto dentro de un elemento más grande, sin afectar la estructura del documento. Se usa comúnmente para aplicar estilos o scripts a partes específicas del texto.
    section Esta etiqueta se utiliza para agrupar contenido temáticamente relacionado dentro de una página. Es útil para dividir una página en secciones distintas, como capítulos en un libro o secciones en un periódico.
    Article Se utiliza para representar contenido independiente y autónomo, como un artículo, una entrada de blog o un comentario. Se distingue de section en que article se puede reutilizar o distribuir de manera independiente del resto del contenido de la página.
    header Se utiliza para definir la cabecera de una sección o de un documento completo. Contiene elementos introductorios o de navegación, como logotipos, encabezados, menús y enlaces de retorno.
    footer Esta etiqueta se utiliza para definir el pie de página de una sección o de un documento completo. Generalmente contiene información de contacto, enlaces relacionados, derechos de autor y otra información similar.
    nav Se utiliza para definir una sección de navegación en una página. Contiene enlaces de navegación que permiten a los usuarios moverse por el sitio web o acceder a diferentes partes de la página actual.
    aside Esta etiqueta se utiliza para definir contenido secundario o complementario que está relacionado con el contenido circundante pero que no es su parte principal. Por ejemplo, barras laterales, notas al margen, anuncios o widgets.
