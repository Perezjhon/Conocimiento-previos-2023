Modelo de cajas

    Modelo de cajas
    Content Es el área donde se muestra el contenido del elemento, como texto, imágenes, etc.
    Padding Es el espacio entre el contenido y el borde del elemento. Se puede establecer usando propiedades como padding-top, padding-right, padding-bottom, padding-left.
    borde Es una línea que rodea el contenido y el relleno del elemento. Se puede establecer usando propiedades como border-width, border-style, border-color.
    Margin (margen) Es el espacio entre el borde del elemento y los elementos adyacentes. Se puede establecer usando propiedades como margin-top, margin-right, margin-bottom, margin-left.
