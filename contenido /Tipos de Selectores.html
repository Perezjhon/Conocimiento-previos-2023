Tipos de selectores

    Tipos de selectores
    Selector Universal ('*') Selecciona todos los elementos de la página.
    Selector de tipo Selecciona todos los elementos que coinciden con un tipo específico, como ('p') para párrafos, ('a') para enlaces, etc.
    Selector de Clase ('.') Selecciona todos los elementos que tienen asignada una clase específica.
    Selector de ID ('#') Selecciona un elemento específico que tiene asignado un identificador único.
    Selector de Atributo Selecciona elementos basados en el valor de sus atributos, como [attribute] para seleccionar elementos que tengan un atributo específico, [attribute="value"] para seleccionar elementos con un atributo y un valor específicos, etc.
    Selector de Clase Anidado Selecciona elementos que son hijos directos de un elemento con una clase específica.
